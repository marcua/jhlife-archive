Title: Rolling Oms Yoga New Summer Class Schedule

I will be running an OPEN level class this summer on Tuesday evenings from 6:30pm-7:45pm (a 75 minute class) starting June 16th and running through July 22nd. Saturday mornings, BASICS Level I, from 9:15am-10:45am
Tuesday evenings, OPEN Level class from 6:30pm-7:45pm
Tuesday evenings, INTERMEDIATE Level II, from 8pm-9:15pm

The class fees will remain the same, with the standard class fee applying whether the class is 75 minutes or 90 minutes. If you have any questions let me know and I'll be glad to answer.
