I'll just admit it.  This is my fault.  JHLife has been swamped by spammers lately.  On many days last year, we were averaging dozens to hundreds of new spam accounts registered per day.   My main tool for reducing this problem has been to block ranges of IP addresses that these automated account registration bots originate from.   I take care not to block IP addresses for any of our legitimate users, but occasionally I can make a mistake.  Or your internet service provider may start using an IP address range that I've blocked previously.   I apologize for any inconvenience this has caused.

If the problem happens again, please email me at calendar (at) jacksonheightslife.com, and I'll try to fix it.   And if you know how, it would be helpful if you can make note of the IP address that is being blocked.

One other thought.  It seems bizarre to me that this problem would only exist with one browser.  I don't know enough about how this stuff works, but it seems like a good precaution to run your antivirus software just to make sure some malware on your computer isn't responsible for this.
