We've been having another ongoing problem with users in good standing being told they are banned from the site.  Presumably, this has been a problem with some new users who are trying to sign up for the site as well.

We have set up some filters designed to keep the spambots from taking over the site, but we think that some of these former spam IP addresses may now be used by legitimate internet service providers, and assigned on a rotating basis to their customers.

In order to reduce the problem of legitimate users being blocked from the site, we have removed a large number of our older bans, and we have labelled each of our remaining bans with a code that should help us more easily trace the problem.   If you try to log in to this site, and receive a ban notice:

  • First, clear any JacksonHeightsLife.com cookies from your browser and try again
  • Contact us at jhlife.help (at) gmail (dot) com to let us know you're having this problem.   Please be use to note the reason given for the ban (this could be an IP range like "23.*.*.*" or a phrase like "US Hostname Required" or a string like "hefExhini98")

Sorry again for any trouble you've been having with the site.

--The Moderators
