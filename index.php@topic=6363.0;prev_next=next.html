3's&4's class(for children born before December 2008) is more active physically as well as verbally to meet children's needs. We'll be introducing solfege (Do, Re, Mi,..etc) in our tonal patterns and we'll be playing rhythm patterns with more specific/detailed body movements. It's fundamentally same but basically more active. It all depends on each case, but some children sing more in class by this age, some still sing only at home. Making music shouldn't be forced, except a few rules to be safe in group-setup, of course, so parents' understanding and participation are still very important, since it has to be fun for all !!!
