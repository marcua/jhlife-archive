Neighborhood Chat / Re: Colombian Election 2010
« on: March 14, 2010, 09:59:05 PM »
the voting process in colombia disenfranchises many.  It's by paper ballot and to vote, you must place an x on the party you are voting for, and then the individual candidate.  if you just put an x on the candidate, your vote doesn't count.  and if you just put an x on the party, you vote for all candidates on that party's line.

Reducing Class Size

Dromm has been a long-time advocate for reducing class size to help raise student achievement, especially in at-risk schools.
