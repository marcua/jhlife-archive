Title: Re: Best Eyebrow Threading for $5!
Post by: Chuckster on January 06, 2008, 09:43:22 PM
Todd, eyebrow threading is a dipilitory (sp?) technique whereby hairs along the eyebrow line are removed using thread and a twisting motion rather than the more conventional methods like waxing or tweezing.  Threading is quicker and if done correctly, beautifully sculpts the eyebrows.  It's a traditional method that is popular in India, perhaps this explains its popularity in Jackson Heights.  I've had this done twice, and on each occasion, my eyebrows looked amazing.
