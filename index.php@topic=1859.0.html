Assessments are generally for capital improvement projects that either were not included in any capital improvements budget and/or cannot be underwritten by existing money in reserve fund. Personally, an assessment for necessary work (like fixing an elevator so that it functions) is OK but an assessment for cosmetic work (like renovating the interior of the elevator cab) is less so. If we're absolutely swimming in money great, but to ask for extra money for cosmetic projects while basic repairs still need to be done should be on a voluntary basis rather than compelled by the board.
