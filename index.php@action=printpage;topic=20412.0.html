Title: Eat Something New in Queens challenge next week
Post by: lsr220 on February 17, 2021, 10:00:59 AM
Hi neighbors,

I'm running a 5-day email challenge next week (starting Feb. 23) that I hope will help people switch up their dining routines, share recs with neighbors, and get inspired to try some new-to-them restaurants/cuisines/dishes. It's called Eat Something New in Queens.

It's a challenge I've run before via my website (Eat Your World), but I've never done it locally, and I think it'll be a lot of fun! Even for those of us who do this sort of thing all the time--this will be different. We'll be doing it together and learning from each other. There's always something new to try!

(Apologies if you've seen this before; I've already posted about it in some JH Facebook groups and on the JHFamilies forum.)

If you're interested, the signup is here: https://adept-thinker-4205.ck.page/6b3effc43d?fbclid=IwAR1Zht9dYuWgSiS_Sg0M5Rq3TaM-nFtFKCn-Ywiv_cnQ3jVtH7NWEmYz1lU

Hope to see you there!
