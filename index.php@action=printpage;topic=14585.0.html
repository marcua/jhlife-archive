Title: Re: In Need of New Boiler
Post by: artfldodger on March 22, 2016, 06:03:31 PM
Same thing happened to mine last year, depending on the size it's going to be in the range you were quoted.  though its always helpful to get other prices.  I'm not sure if I would reccomened who I used.  The work is great but communication is really tricky, unless you know Polish.   

Other things to note:
- The installer will typically match the new boiler BTU with the old one.  But if you wanted to check how much BTU you really need this site has a great excel sheet to help size the boiler based on your existing radiators - http://www.crownboiler.com/Support/Education/Boiler-Sizing
- Use http://www.supplyhouse.com/ for looking for boiler specs (if you're so inclined).  I ended up using Weil-McClain, good efficency and reviews
- National Grid (or your gas supplier) usually has sizable rebates if you get a boiler about set % efficiency, check their website
